What's inside

Everything you need to run a professional VA operation

Six fully integrated modules, one secure cloud-synced desktop app.

AI Assistant

Ask questions, update data, and get intelligent insights using your own API key. Supports Google Gemini, Anthropic Claude, and Alibaba DashScope.

Competency Tracker

Build, score, and track competency documents across multiple domains. Full BARS rubric support with L1–L4 behavioural anchors per competency.

Boss's Queue

A structured review pipeline for submitting competency documents for approval. Track status, priority, and feedback notes end-to-end.

Hours & Pay Tracker

Log work sessions with start/end times, calculate earnings automatically, and export monthly reports. Supports 10 currencies including USD, GBP, KWD, and more.

Document Store

Version-controlled storage for all competency documents and scoring templates. Track status, link URLs, and maintain a full audit trail.

Masterplan

Define and track your own custom project phases. Add phase names, key outputs, and advance status as work progresses — fully flexible for any team structure.
